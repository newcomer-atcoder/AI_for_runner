# ランニングサポートAI
## ■アプリの紹介
このアプリは、ランニングで **「実際に走る距離(km) 」** を決めるアプリです。  
過去のランニングの記録を基に、AIが **「実際に走る距離(km) 」** の目安を提示してくれます。  

### 開発契機：  
<details>
  <summary>開発契機を表示する</summary>
  <p>
  　このアプリは、学生時代に陸上長距離/マラソンを経験した開発者が、ダイエット目的でランニングを再開した時に作成したものです。<br/>
　開発者がランニングを再開した当初、以下の様に「実際に走る距離(km) 」の予定と実績が嚙み合わないことが多々あり<br/>
　原因はその日の体調を考慮できていないためと分析しました。<br/>
　　ex.<br/>
　　　10キロ走ろうと思ったが、「調子が出ず」7キロで終わった<br/>
　　　5キロだけ走る予定だったが、「体調がよかったので」7キロ走った<br/><br/>
　上記を解決するため、ランニングの前に「体調」から「実際に走る距離(km) (=実績の予想)」を予想して<br/>
　この予想を目安にランニングできればと考え、このアプリを作成しました。<br/>
  </p>
</details>

## ■アプリの使い方
(2026/03 時点)  
GitHubよりこのリポジトリをcloneした後、以下手順に従いご利用ください。  
Python3.10~3.12が使用できる環境であればお使いいただけます。  
またこのアプリはオフラインでご利用いただけます。

### アプリ実行手順 
<details>
  <summary>アプリ実行手順を表示する</summary>
  <ol>
    <li><b>src\main.py</b>を実行する。</li><br/>
    <li>5秒程待つとブラウザが自動で起動する。画面の以下ラジオボタンから項目を選び、「次へ進む」ボタンをクリックする。
　　　　　　<br/>「y：ランニング記録の入力に進む」を選択：手順3.に進む
　　　　　　<br/>「n：AIが本日の走行距離(km)を算出します」を選択：手順6に進む
　　　　　　<br/>※初回起動の場合は、強制的に手順3.に進む
    </li><br/>
    <li>表示された画面に従い「過去のランニング記録」を入力する。
        <br/>以下の4項目を入力する。
        <ul>
          <li>日付(yyyy/mm/dd)</li>
          <li>走行予定の距離(km) </li>
          <li>体調(%)</li>
          <li>実際に走った距離(km)</li>
        </ul>
        <details>
          <summary>※各項目の補足説明はこちら</summary>
          <ul>
            <li>「走行予定の距離(km)」：ランニングする前に、走りたい距離を決めておく。</li>
            <li>「体調(%)」：その日の調子・体調をフィーリングで入力する。<br/>開発者の場合、平常時を60%として調子が悪ければ50%、良ければ80%などと定めています。</li>
          </ul>
        </details>
    </li><br/>
    <li>「登録」ボタンをクリックして、「◆登録メッセージ」に"登録成功："などと表示されることを確認する。</li><br/>
    <li>すべての記録を入力し終えた後、「登録終了」ボタンをクリックをして登録を終了する。<br/>5秒程度待つと「実際に走る距離(km) 」を推論する画面に遷移する。</li><br/>
    <li>表示された画面に従い「走行予定の距離(km)」「体調(%)」を入力する。<br/>「AI実行」ボタンをクリックすると「◆AIの回答」欄に"本日のあなたの適正距離(km)"を、"実際に走る距離(km) "として表示する。</li><br/>
    <li>「アプリ終了」ボタンをクリックしてアプリの利用を終了する。</li><br/>
  </ol>
</details>

## ■使用環境(OS/言語バージョン etc)
- OS情報：Windows11 x64ベースシステム
- エディタ：VSCode(推奨)、Spyder(Anaconda)(=conda環境を構築)

  - sqlalchemy
