# ランニングサポートAI
## ■アプリの紹介
このアプリは、ランニングで「今日何キロ走るか」を決めるアプリです。  
過去のランニングの記録を基に、AIが「今日何キロ走るか」の目安を提示してくれます。  

### 開発契機：  
<details>
  <summary>開発契機を表示する</summary>
  <p>
  　このアプリは、学生時代に陸上長距離/マラソンを経験した開発者が、ダイエット目的でランニングを再開した時に作成したものです。<br/>
　開発者がランニングを再開した当初、以下の様に「今日何キロ走るか」の予定と実績が嚙み合わないことが多々あり<br/>
　原因はその日の体調を考慮できていないためと分析しました。<br/>
　　ex.<br/>
　　　10キロ走ろうと思ったが、「調子が出ず」7キロで終わった<br/>
　　　5キロだけ走る予定だったが、「体調がよかったので」7キロ走った<br/><br/>
　上記を解決するため、ランニングの前に「体調」から「今日何キロ走るか(=実績の予想)」を予想して<br/>
　この予想を目安にランニングできればと考え、このアプリを作成しました。<br/>
  </p>
</details>

## ■アプリの使い方
(2026/03 時点)  
GitHubよりこのリポジトリをcloneした後、以下手順に従いご利用ください。  
Python3.10~3.12が使用できる環境であればお使いいただけます。  
またこのアプリはオフラインでご利用いただけます。

### アプリ実行手順 
1. **\src\main.py**を実行する。
2. エクスプローラが自動で起動した後、AIに学習させるデータ(=過去のランニングの記録)を選択する。
3. コンソールに従い入力し、AIにあなたのランニングの目標(=何キロ走るか)を回答してもらう。
